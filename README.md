# dans.computer 🤖

> *A digital playground where calendars meet bananas and wizards write essays*

Welcome to the most over-engineered personal website you'll encounter today. Built with Next.js because apparently we need React to display text on a screen.

## What Even Is This?

- **meet.danfarr.com** - A Calendly clone that creates Google Meet links. Yes, I reinvented the wheel, but this wheel has *style*.
- **ScrumBananas** - A game where bananas have more functionality than most enterprise software
- **Essays** - Words arranged in sentences. Some even have parallax backgrounds!
- **Sand Wizards** - They generate terrain and probably judge you for your CSS choices

## Getting Started (If You Must)

```bash
pnpm dev  # Starts the dev server, summoning localhost:3000 from the ether
```

Open [http://localhost:3000](http://localhost:3000) and witness the chaos.

## The Tech Stack (Overkill™ Edition)

- **Next.js** - Because plain HTML would be too simple
- **Supabase** - PostgreSQL with extra steps
- **NextAuth** - Authentication so users can... read my essays?
- **Google Calendar API** - To remind me I have no meetings
- **Framer Motion** - Animations because static content is *boring*
